Название проекта: AETHEL

* * * * * *
приложение для прослушивания ASMR и медитативных треков с возможностью онлайн-стриминга и оффлайн-воспроизведения. Приложение предоставляет удобный интерфейс для работы с библиотекой расслабляющих звуков, поддерживает интеллектуальный таймер сна и позволяет скачивать треки для прослушивания без интернета.
* * * * * *


Основные функции:
+ Звуковое Ядро
	- ASMR-зоны по категориям
	- Онлайн-стриминг
	- Оффлайн-библиотека
	- Таймер сна
+ Управление Контентом
	- Категоризация - удобная навигация по типам звуков​​
	- Персональная библиотека - управление сохраненными треками
	- Очередь загрузки - фоновое скачивание с отслеживанием прогресса

Навигация:
- Стартовый экран: Главный экран (ASMR Screen)
- Переход к категориям: Нажатие на кнопку "Категории" или карточку категории
- Переход к трекам: Выбор категории из списка категорий
- Воспроизведение: Нажатие на трек из списка (онлайн-плеер)
- Скачивание: Кнопка загрузки на карточке трека (добавление в очередь)
- Библиотека: Нижняя панель навигации (вкладка "Библиотека")
- Оффлайн-воспроизведение: Выбор скачанного трека из библиотеки
- Таймер сна: Кнопка таймера оффлайн плеере


Сценарии использования:
- Вечерний сеанс релаксации:
Пользователь открывает приложение и переходит в категорию "Сон". Выбирает трек "Дождь в ночном лесу" и запускает воспроизведение. Устанавливает таймер на 30 минут с плавным затуханием. Засыпает под звуки дождя, музыка автоматически выключается.

- Подготовка к путешествию:
Пользователь открывает категорию "Медитация". Скачивает 5 любимых треков для оффлайн-доступа через кнопку загрузки. Треки появляются в библиотеке. В самолёте использует оффлайн-плеер без интернета.

- Рабочая концентрация:
Пользователь выбирает категорию "Фокус" на главном экране. Запускает трек "Белый шум для работы" в онлайн-плеере. Работает 60 минут под фоновый звук. Повторяет трек в цикле через настройки плеера.

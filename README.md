Название проекта: Aethel

* * * * * *
Эмоционально-глубокое медитативное приложение с акцентом на атмосферу и стиль. Приложение сочетает бинуральные ритмы, ASMR-зоны, визуальное погружение и интеллектуальный трекинг настроения для создания персонализированного опыта релаксации.
* * * * * *


Основные функции:
+ Звуковое Ядро 
	- Настраиваемые звуковые миксы: комбинации природных звуков (дождь, гром, белый шум)
	- Бинуральные ритмы: отдельные разделы для фокуса, релаксации и сна
	- ASMR-зоны: звуковые ландшафты ("Ночной лес", "Кофейня утром", "Библиотека")
	- Таймер для засыпания: плавное затухание звуков к концу сеанса
+ Визуальное Погружение
	- Интерактивные абстрактные сцены: плавающие частицы, реагирующие на звук
	- Режимы затемнения с едва заметными элементами
+ Трекер Настроения
	- Дневник эмоций: быстрая оценка состояния до и после сеанса
	- Визуальная статистика: графики влияния звуков на настроение
	- Персональные сборники: автоматически формируемые плейлисты

Навигация:

Стартовый экран: Главный экран
Переход к звуковым сценам: Нажатие на любую звуковую сцену
Переход к статистике: Нажатие на блок "Моя статистика"
Дневник настроения: Автоматически появляется при запуске приложения
Настройки: Нижняя панель навигации


Сценарии использования:

Вечерний сеанс релаксации:
- Пользователь открывает приложение, отмечает текущее настроение как "Тревога"
- Приложение предлагает сборник "Уютные звуки" на основе истории
- Пользователь запускает сеанс, экран переходит в режим "Тёмный лес"
- Через 20 минут приложение мягко уведомляет о завершении
- Пользователь отмечает итоговое настроение как "Спокойствие"
- Данные сохраняются в статистику

Быстрый перерыв на фокусировку:
- Пользователь выбирает бинуральные ритмы для концентрации
- Запускает 15-минутный сеанс с таймером
- Работает с плавающими визуальными элементами для концентрации
- По окончании возвращается к работе
